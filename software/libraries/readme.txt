This directory contains the libraries used.
They have to be placed in arduino library directory (eg C:\Users\youruser\Documents\Arduino\libraries)

LiquidCrystal
-------------
New Liquid Crystal
by F. Malpartida - fmalpartida@gmail.com
website https://bitbucket.org/fmalpartida/new-liquidcrystal/wiki/Home

RTCLib
------
From: https://github.com/adafruit/RTClib.git
(there is another library but I've not used it http://smart-prototyping.com/Arduino-Tiny-RTC-I2C-real-time-clock-module-24C32-storage-DS1307.html)

OneWire
-------
Version 2.2 from Paul Stoffregen's site http://www.pjrc.com/teensy/td_libs_OneWire.html
Arduino's OneWire page http://playground.arduino.cc/Learning/OneWire (warning: has buggy version) 

DallasTemperature
-----------------
From: https://github.com/milesburton/Arduino-Temperature-Control-Library.git
By milesburton http://milesburton.com/Main_Page?title=Dallas_Temperature_Control_Library
